This demo was originally from Srinivas Tamada's 9lessons.info site. The original article is located here:
http://www.9lessons.info/2013/09/angularjs-tutorial-two-way-data-binding.html

His original code only gets the list of comments when the page loads. It then updates the list when the user adds or deletes an item without syncing it back up with the database. My changes now allows the code to sync up the list with the database every time a user adds or deletes a comment. I only had to make a few minor updates the comments.php and the controller's AngularJS code.
